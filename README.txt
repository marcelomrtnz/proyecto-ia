Introducción  

En el marco del curso de Inteligencia Artificial, presentamos nuestro proyecto titulado "Predicciones de Inundaciones". Este proyecto 
tiene como objetivo desarrollar un modelo de inteligencia artificial capaz de predecir inundaciones utilizando datos históricos de 
eventos de inundaciones recientes, con los cuales se elabora un modelo predictivo.  

El objetivo principal de este proyecto es crear un sistema predictivo que permita anticipar eventos de inundaciones en ciertas zonas. 
Entendemos que el modelo sigue siendo altamente probabilístico, ya que existen múltiples variables que pueden cambiar con el tiempo, 
así como las condiciones climáticas.

Metodología  

Para lograr este objetivo, hemos recopilado y analizado datos históricos de inundaciones. Nos estamos basando en detalles generales 
encontrados en artículos científicos y con los cuales concluimos que la forma de relacionar las predicciones de las inundaciones es 
validar los niveles de los cauces de los ríos, ya que estos tienen relación directa con las inundaciones. Utilizando técnicas avanzadas 
de aprendizaje automático y análisis de datos, estamos desarrollando un modelo predictivo que pueda identificar patrones y tendencias 
en los datos históricos. 

Buscamos información a nivel regional y encontramos que localmente no existe información relevante sobre el tema, encontrándose que 
Panamá es el país más cercano con elementos como información histórica, accediendo a ella, además tienen información en tiempo real. 
Este modelo nos permite establecer un marco de trabajo para poder aplicarlo en Honduras y demuestra la 
importancia de tener y dar acceso a una data abierta.  

Resultados Esperados 
 
Esperamos que nuestro modelo de predicción de inundaciones sea capaz de:  
- Identificar áreas de alto riesgo de inundación.  
- Predecir la probabilidad de ocurrencia de inundaciones en diferentes regiones.
